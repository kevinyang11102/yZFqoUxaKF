# 养老平台管理系统 java392

## 项目概述

养老平台管理系统是一个基于Springboot和Vue的前后端分离的Web应用。该系统旨在提供用户友好的管理界面，实现用户和管理员的登录功能，以支持养老机构的日常运营管理。

## 技术栈

- 后端：Java、Springboot、MyBatis、Maven
- 前端：Vue
- 数据库：MySQL 5.7

## 功能模块

- 用户登录与权限验证
- 管理员登录与权限验证
- 基础信息管理
- 日志记录与监控

## 系统角色

- 普通用户
- 管理员

## 运行环境

- JDK 1.8
- Maven
- MySQL 5.7

## 部署步骤

1. 准备所需的环境，包括JDK、Maven、MySQL。
2. 创建数据库，导入数据结构。
3. 编译后端代码，打包。
4. 部署前端代码到Web服务器。
5. 配置服务器，启动后端服务。
